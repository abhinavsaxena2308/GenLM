# GenLM

GenLM is a modern, AI-powered talent acquisition platform built with React, TypeScript, and Tailwind CSS. It features a stunning, dynamic hero section with a looping video background, smooth animations, and a responsive design.

## Features

- **Dynamic Hero Section**: Features a high-quality video background with smooth fade transitions and a looping effect.
- **Modern UI**: Built with Tailwind CSS and Radix UI components for a premium look and feel.
- **Responsive Design**: Optimized for various screen sizes, from mobile to large desktops.
- **Navigation**: Includes a sticky navigation bar with smooth scrolling and hover effects.
- **Social Proof**: A dedicated section to showcase testimonials and build trust.

## Tech Stack

- **Frontend**: React 18, TypeScript
- **Styling**: Tailwind CSS, Radix UI
- **Build Tool**: Vite
- **Testing**: Playwright, Vitest

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd GenLM
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the development server:

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view it in your browser.

### Build

Build the application for production:

```bash
npm run build
```

### Testing

Run the test suite:

```bash
npm test
```

Run end-to-end tests with Playwright:

```bash
npx playwright test
```
